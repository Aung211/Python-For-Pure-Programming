### 0x02 Hello World Program

##### Example 1
```python
print("Hello World")
```

###### Output:
```
Hello World
```

##### Example 2

```python
print(1+2)
```

###### Output:
```
3
```

##### Example 3

```python
print("Logix","Owl")
```

###### Output:
```
Logix Owl
```

##### Example 4

```python
print("Logix",6+9)
```

###### Output:
```
Logix 15
```

##### Example 5

```python
print("Hello","Nay","Kg","Lr?")
```

###### Output:
```
Hello Nay Kg Lr?
```

##### Example 6

```python
print("Hello")
print("Nay Kg Lr?")
```

###### Output:
```
Hello
Nay Kg Lr?
```

##### Example 7

```python
print("Hello",end=' ')
print("Nay Kg Lr?")
```

###### Output:
```
Hello Nay Kg Lr?
```
